## 🎨 UI 改进建议

### P0 - 必须实现（核心价值）

#### 1. 深色主题切换
**参考：** n8n
**改进：**
- 提供暗色/亮色主题切换
- 默认深色（专业感强）
- 配色方案：
  - 背景：`#1a1a2e` (深空蓝)
  - 卡片：`#16213e` (深蓝)
  - 强调色：`#e94560` (珊瑚红)
  - 成功：`#0f3460` → `#4ade80` (绿色)

#### 2. 节点样式优化
**参考：** n8n + ComfyUI
**改进：**
```css
/* 当前问题 */
- 节点样式单一
- 无类型区分
- 状态指示不明确

/* 改进方案 */
.node-card {
  border-radius: 8px;        /* 圆角 */
  box-shadow: 0 2px 8px rgba(0,0,0,0.15); /* 阴影 */
  border-left: 4px solid;    /* 左侧颜色条区分类型 */
  padding: 12px;
  transition: all 0.2s;
}

/* 类型颜色编码 */
.node-python { border-color: #306998; }  /* Python 蓝 */
.node-http { border-color: #00c7b7; }    /* HTTP 青 */
.node-condition { border-color: #f59e0b; } /* 条件橙 */
.node-llm { border-color: #8b5cf6; }     /* LLM 紫 */

/* 状态指示 */
.node-running { animation: pulse 1s infinite; }
.node-success { border-color: #10b981; }
.node-error { border-color: #ef4444; }
```

#### 3. 连线设计优化
**参考：** ComfyUI
**改进：**
- 使用贝塞尔曲线（非直线）
- 数据流动画（执行时）
- 端口颜色编码（输入/输出）
- 连线粗细区分数据类型

```css
/* 连线样式 */
.flow-edge {
  stroke-width: 2px;
  stroke: #64748b;
  fill: none;
}

/* 执行时动画 */
.flow-edge.running {
  stroke-dasharray: 10;
  animation: dash 1s linear infinite;
}

@keyframes dash {
  to { stroke-dashoffset: -20; }
}
```

#### 4. 配置面板改进
**参考：** n8n
**改进：**
- 右侧抽屉式（非弹窗）
- 表单分组折叠
- 代码编辑器行号显示
- 实时参数验证提示
- 保存状态指示（自动保存）

#### 5. 执行状态可视化
**参考：** n8n + Dify
**改进：**
```
节点状态指示：
┌─────────────────┐
│ 🐍 Python 脚本   │ ← 图标 + 名称
├─────────────────┤
│ ● 运行中...     │ ← 状态指示灯
│   2.3s          │ ← 执行时间
└─────────────────┘

状态颜色：
- 等待：灰色 (#94a3b8)
- 运行：蓝色闪烁 (#3b82f6)
- 成功：绿色 (#10b981)
- 失败：红色 (#ef4444)
```

### P1 - 建议实现（增强体验）

#### 6. 画布交互优化
**改进：**
- 鼠标滚轮缩放（25%-200%）
- 拖拽空白区域平移
- 框选多个节点
- 自动布局按钮
- 小地图导航（大工作流）
- 网格吸附辅助

#### 7. 节点快捷操作
**改进：**
- 双击节点编辑
- 右键菜单（复制/删除/折叠）
- 快捷键支持：
  - `Delete` - 删除选中
  - `Ctrl+C/V` - 复制粘贴
  - `Ctrl+Z/Y` - 撤销重做
  - `Space+ 拖拽` - 平移画布
  - `Ctrl+ 滚轮` - 缩放

#### 8. 空状态设计
**改进：**
```
当前：空白画布，无指引

改进：
┌──────────────────────────────┐
│                              │

│                              │
│   从左侧拖拽节点到画布       │
│   或点击"+"快速添加          │
│                              │
│   [查看模板库] [观看教程]    │
│                              │
└──────────────────────────────┘
```

#### 9. 错误提示优化
**改进：**

## 📐 设计规范建议

### 配色方案
```css
:root {
  /* 主色调 */
  --primary: #3b82f6;
  --primary-hover: #2563eb;
  
  /* 功能色 */
  --success: #10b981;
  --warning: #f59e0b;
  --error: #ef4444;
  --info: #3b82f6;
  
  /* 中性色 */
  --bg-dark: #1a1a2e;
  --bg-card: #16213e;
  --text-primary: #f1f5f9;
  --text-secondary: #94a3b8;
  
  /* 节点类型色 */
  --node-python: #306998;
  --node-http: #00c7b7;
  --node-condition: #f59e0b;
  --node-llm: #8b5cf6;
}
```

### 间距规范
```css
/* 基础间距单位：4px */
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 16px;
--spacing-lg: 24px;
--spacing-xl: 32px;
```

### 圆角规范
```css
--radius-sm: 4px;   /* 按钮/输入框 */
--radius-md: 8px;   /* 卡片/节点 */
--radius-lg: 12px;  /* 大卡片 */
--radius-full: 9999px; /* 圆形 */
```

### 阴影规范
```css
--shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
--shadow-md: 0 4px 6px rgba(0,0,0,0.1);
--shadow-lg: 0 10px 15px rgba(0,0,0,0.15);
--shadow-xl: 0 20px 25px rgba(0,0,0,0.2);
```
