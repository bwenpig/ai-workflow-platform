## ✅ 测试清单

### 测试 1：节点添加
1. 打开应用
2. 查看左侧"节点工具箱"
3. 确认显示"🐍 Python 脚本"按钮
4. 点击按钮
5. **预期结果**: 画布上出现紫色渐变的 Python 节点

### 测试 2：节点外观
1. 观察新添加的 Python 节点
2. **预期结果**:
   - 背景：紫色渐变（#667eea → #764ba2）
   - 边框：Python 蓝色（#306998）
   - 图标：🐍
   - 文字：Python 脚本
   - 左右两侧有连接点（输入/输出端口）

### 测试 3：配置面板打开
1. 点击画布上的 Python 节点
2. 查看右侧信息面板
3. **预期结果**: 显示 Python 脚本配置面板

### 测试 4：Monaco Editor 编辑器
1. 在配置面板中查看代码编辑器
2. **预期结果**:
   - 编辑器高度约 300px
   - 深色主题（vs-dark）
   - 显示行号
   - 无代码时显示空白

### 测试 5：Python 语法高亮
1. 在编辑器中输入以下代码：
   ```python
   import requests
   
   def fetch_data(url):
       response = requests.get(url)
       return response.json()
   
   result = fetch_data("https://api.example.com/data")
   print(result)
   ```
2. **预期结果**:
   - `import`, `def`, `return`, `print` 显示为紫色（关键字）
   - 字符串显示为黄色
   - 函数名显示为蓝色
   - 注释显示为绿色（如果有）

### 测试 6：代码编辑
1. 在编辑器中编写任意 Python 代码
2. 修改代码内容
3. **预期结果**: 代码可以正常编辑，光标移动正常

### 测试 7：配置项测试
#### 脚本描述
1. 在"脚本描述"输入框中输入文字
2. **预期结果**: 可以正常输入和编辑

#### 超时时间
1. 修改超时时间（例如：60 秒）
2. **预期结果**: 可以输入 1-300 之间的数字

#### 依赖包
1. 在依赖包文本框中输入：
   ```
   requests
   pandas
   numpy
   ```
2. **预期结果**: 可以输入多行，每行一个包名

### 测试 8：保存配置
1. 填写脚本描述
2. 编写 Python 代码
3. 设置超时时间
4. 添加依赖包
5. 点击"保存配置"按钮
6. **预期结果**: 
   - 显示成功提示消息
   - 配置保存到节点

### 测试 9：配置持久化
1. 保存配置后，点击其他节点
2. 再次点击 Python 节点
3. **预期结果**: 之前配置的内容仍然存在

### 测试 10：取消操作
1. 修改配置但不保存
2. 点击"取消"按钮
3. **预期结果**: 配置恢复为上次保存的值

## 🐛 问题排查

### 问题 1：看不到 Python 节点按钮
**可能原因**: 应用未重新加载
**解决方案**: 刷新浏览器页面（Cmd+R）

### 问题 2：编辑器不显示
**可能原因**: Monaco Editor 未正确加载
**解决方案**: 
1. 检查浏览器控制台是否有错误
2. 确认 `@monaco-editor/react` 已安装
3. 重启开发服务器

### 问题 3：语法高亮不工作
**可能原因**: 语言设置错误
**解决方案**: 确认编辑器 `language="python"`

### 问题 4：配置无法保存
**可能原因**: 节点状态未更新
**解决方案**: 
1. 检查浏览器控制台错误
2. 确认 `onSave` 回调正确调用
3. 验证节点数据更新逻辑
