## 🔍 失败原因分析

**主要问题：** 前端页面元素未正确加载

**错误信息：**
```
Error: locator.dragTo: Test timeout of 60000ms exceeded.
Call log:
  - waiting for getByText(/Python 脚本/)
```

**根本原因：**
1. 节点面板未正确渲染
2. "Python 脚本"节点文本与页面实际内容不匹配
3. 前端可能需要更长的加载时间
4. 测试环境与手动操作环境可能存在差异
